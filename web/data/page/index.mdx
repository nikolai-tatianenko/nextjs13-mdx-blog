# Markdown Basics

This is an example of a Markdown document with various basic components.

## Headers

You can use headers to structure your document. There are six levels of headers in Markdown:

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6

## Text Formatting

You can apply formatting to your text using different Markdown syntax:

- _Italic_ text can be written using asterisks or underscores.
- **Bold** text can be written using double asterisks or double underscores.
- ~~Strikethrough~~ text can be achieved by using double tildes.
- You can create `inline code` by enclosing the code snippet within backticks.

## Lists

You can create ordered and unordered lists:

- Unordered List
- Nested item
- Another nested item
- Ordered List

1. First item
2. Second item
3. Third item

## Links

You can create links in Markdown:

- [https://localhost.com](https://localhost.com) - Hyperlink to localhost

## Images

You can embed images in Markdown using the following syntax:

![Github logo](https://github.githubassets.com/images/modules/site/icons/footer/github-mark.svg)

## Blockquotes

You can create blockquotes to highlight quotes or important information:

> This is a blockquote.
> It can span multiple lines.

## Code Blocks

You can display code blocks in Markdown:

```javascript
alert('Hello, Markdown!');
```

## Horizontal Rule

You can insert a horizontal rule to separate sections:

---

## Tables

You can create tables in Markdown:

| Name   | Age | Country   |
| ------ | --- | --------- |
| John   | 28  | USA       |
| Emily  | 32  | Canada    |
| Miguel | 24  | Argentina |

## Task Lists

You can create task lists:

- [x] Task 1
- [ ] Task 2
- [ ] Task 3

## Conclusion

These are some basic components of Markdown. There are many more features and extensions available for Markdown, but this should give you a good starting point.

Feel free to explore and experiment with Markdown to create rich and formatted documents!
